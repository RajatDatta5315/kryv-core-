# KRYV Core

KRYV Core is a Next.js 14 application built with Tailwind CSS, featuring a custom design and architecture.

## Getting Started

To get started with KRYV Core, clone the repository and install the dependencies:
```bash
npm install
```
## Development

To start the development server, run:
```bash
npm run dev
```
## Production

To build and start the production server, run:
```bash
npm run build
npm run start
```
