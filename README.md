# KRYV Core

KRYV Core is a Next.js 14 application built with Tailwind CSS. It serves as the foundation for our web applications, providing a robust and scalable architecture.

## Features

* Production-grade Next.js 14 codebase
* Tailwind CSS for styling and layout
* Custom, hand-crafted UI components

## Getting Started

To get started with KRYV Core, clone the repository and run the following commands:

```bash
npm install
npm run dev
```
